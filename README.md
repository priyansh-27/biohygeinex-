BioHygieneX

BioHygieneX is a smart sanitary and medical waste management platform developed to encourage safe waste disposal and improve hygiene practices.

The platform allows users to scan QR codes on disposal bags, earn reward points for responsible waste disposal, track waste collection vehicles, and raise complaints related to waste management services.

Features

QR code-based waste disposal tracking
Reward points system for users
Live waste collection vehicle tracking
Complaint registration and tracking
Analytics dashboard
Educational content on hygiene and waste management

TECHNOLOGIES USED

HTML
CSS
JavaScript
Python
FastAPI
MySQL
Netlify

PROJECT GOAL

The goal of this project is to promote proper sanitary and medical waste disposal while increasing community participation through a simple and transparent digital platform.
